# pot8o-chip
A Chip8 Emulator written in C++

# Keybindings
keypad:

0-9

A: /

B: *

C: -

D: +

E: enter

F: .

pause: P

play: G

unlock clock speed: U

lock clock speed: L

change game: C

exit: escape
